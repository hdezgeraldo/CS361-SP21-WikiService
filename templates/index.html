Welcome to my Image Scrapper

Description: You can access my program by passing your GET or POST parameters via the /image-scrapper page on this web app. This app will find the FIRST poster or thumbnail image in a given Wikipedia article, and will return its HTTP image URL.

GET method

  • After the URL web address has ended, we include a question mark (?) followed by the name/value pairs:
  • Make sure that the "key" is "url", and that your "value" is the Wikipedia URL
  • Don't forget to replace the URL example with the actual URL for this web app

https://mypyimagescrapper.ue.r.appspot.com/image-scrapper?url=https://en.wikipedia.org/wiki/Oregon_State_University

JSON formatted Response

{ "imageURL": "//upload.wikimedia.org/wikipedia/en/8/8a/generic_image.jpg" }

POST method

  • You can make a JSON POST request to this web app's /image-scrapper page
  • Make sure that the "key" is "url", and that your "value" is the Wikipedia URL

{ "url": "https://en.wikipedia.org/wiki/Oregon_State_University" }

JSON formatted Response

{ "imageURL": "//upload.wikimedia.org/wikipedia/en/8/8a/generic_image.jpg" }

Note: Don't forget to access the app within the /image-scrapper page
